Comment rejoindre la course

Trouver le serveur des Cellules Rapaces

Dans la section Multijoueur de DCS World,

faites une recherche pour trouver le serveur des Cellules Rapaces.

Version de jeu

D'une manière générale, la version de jeu pour le défi est la dernière version DCS OpenBeta disponible.

L'Integrity Check est activé afin d'assurer la validité des temps produits.

Il est possible d'utiliser une livrée avion personnalisée. C'est même vivement encouragé pour rendre les replays vidéos sympas à regarder et permettre de distinguer les joueurs les uns des autres.
Pense à nous envoyer le fichier de ta livrée via notre Discord.
Nous réaliserons un pack spécifique des livrées utilisées dans le défi.

Si vous ne voyez pas le serveur

Il est possible que le serveur soit hors ligne pour maintenance.

En cas de doute ou difficulté, n'hésite pas à venir poser tes questions sur le Discord des Cellules Rapaces.
